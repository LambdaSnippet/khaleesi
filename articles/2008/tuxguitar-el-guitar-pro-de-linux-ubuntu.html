TuxGuitar, el Guitar Pro de Linux (Ubuntu)

2008-07-27

Algunos de los pocos software que windows tenia y linux no era un software de gestión de partituras y tablaturas de instrumentos sintetizados como lo es en el caso de windows el fabuloso GuitarPro.

Pero no hasta hace mucho un equipo de desarrolladores músicos crearon una versión alternativa y libre a guitar pro en linux llamada TuxGuitar (en honor a nuestro tuxito).

Para instalarla solo basta tener algunos paquetes en nuestro sistema, TuxGuitar esta desarrollado en java, por lo que evidentemente tendremos que tener instalada nuestra maquina virtual.

Instalamos nuestra maquina virtual: sudo aptitude install sun-java6-jre Ahora los paquetes necesarios para ejecutar TuxGuitar. sudo aptitude install libswt3.2-gtk-jni libswt3.2-gtk-java Y por ultimo, TuxGuitar. sudo aptitude install tuxguitar Ahora a disfrutar de nuestro TuxGuitar.

Nota: Por cierto, podemos abrir archivos de guitar pro en TuxGuitar desde la versión 3 hasta la actual (version 5).
