Consejos y manuales para el diseño web

2008-02-26

Hoy por la mañana, un amigo de escuela (Gera) me pidió que recomendara algunos manuales que le ayudarán a iniciarse en todo eso del diseño web que es así como lo llamo yo (y no programación web como la materia que tomamos, pero este es punto y aparte), y me acorde como empece a buscar información sobre todo lo que me pudiese aportar conocimiento en el tema.

Sabia de la existencia de HTML, había echo algunas pruebas yo mismo con ejemplos de sitios webs que visitaba, y es mas, había echo una pagina de ejemplo cuando tenia instalado Dreamweaver.

Pero también sabia que esa no era la manera correcta de hacerlo, pronto me entere de la existencia de herramientas, o mejor dicho, tecnologías que hacían del diseño web una potente aplicación de desarrollo, así como de los estándares que debíamos seguir para darle un mejor uso, y flexibilidad a nuestros desarrollos.

Bien, pues ahora puedo decir, Gera, esto es lo que yo te recomiendo.

  • Olvidarse de herramientas como Dreamweaver que solo te harán mas penosa tu aplicación por generación de código basura.
  • Existen herramientas como Quanta Plus (del tipo WYSIWYG )que no genera ese código basura.
  • Sin embargo, como recomendación usar un simple editor de textos, claro, hay algunos que te ayudan bastante como Notepad++ para Windows, o Scite para Linux.
  • Buscar, explorar, indagar en los sitios que te llaman la atención preguntándonde, ¡ aaah, como hizo para hacer eso !? y mirando en sus entrañas y poder ver realmente como fue que hizo posible tal cosa.
  • Paciencia y entrega, que mas.

En cuanto a manuales, aquí solo pongo algunos que me ayudaron a mi y me siguen ayudando a conocer mas sobre esto. Y también acordeones.

  • Cheat Sheet XHTML , XHTML reference
  • Manual de PHP , Cheat Sheet de PHP
  • Manual de Javascript, Cheat Sheet de Javascript
  • CSS (con la especificación basta) , Cheat Sheet de CSS
  • HTML DOM Reference
  • AJAX
