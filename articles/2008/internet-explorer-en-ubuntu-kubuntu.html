Internet Explorer en Ubuntu (Kubuntu)

2008-02-02

Así como lo oyen, instalar Internet Explorer en Linux , y se han de preguntar, para que demonios instalo esa cosa en Linux, bien, pues pueden existir varias razones, por ejemplo, existe un sitio en el que debes entrar y hacer algo importante, pero el único inconveniente que tienes es no poder hacerlo desde Firefox y tu amas Firefox y solo usas Linux como sistema operativo, entonces recures a la vergonzosa tarea de instalar windows para solo poder usar internet explorer. Bien, pues hay otra alternativa no menos vergonzosa que es instalarlo en Linux.

El proceso siguiente es para Kubuntu (obviamente para Ubuntu).

Bien, la herramienta que utilizaremos se llama ie4linux (el 4 es por el parecido en la pronunciación de four con for) y la puedes descargar de aquí .

Ahora solo vamos a instalar cabextract y wine que son requisitos para ie4linux. $ sudo aptitude install wine cabextract Para instalarlo tendremos que descomprimir ie4linux. tar zxvf ies4linux-latest.tar.gz Entramos al nuevo directorio que se creo al descomprimirlo. cd ies4linux-* Y ejecutamos el script. ./ies4linux

Ahora solo tenemos que seguir el proceso de instalación que es decirle que versión queremos, donde queremos que se instale, y si queremos que instale también los plugins de flash.

Y por ultimo, cuando por fin se descargue y se instale, podremos ejecutarlo desde el icono que se creo en el escritorio que se me muy feo, o simplemente haciendo:

$ ie6

También vale la pena decir que ie4linux es un proyecto como muchos otros de código abierto y que el creador hace una atenta invitación de colaborar reportando bug's, sugerencias, parches, y de igual manera si puedes una pequeña donación.
