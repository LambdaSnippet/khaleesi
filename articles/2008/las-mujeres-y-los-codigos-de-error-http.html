Las mujeres y los códigos de error HTTP

2008-02-08

No soy el primero que las re-posteo, hace mucho que ya las había leído y ahora que tengo oportunidad aquí las pongo, no existe mejor forma que estas de describir cada código.

  • Error HTTP 400: Bad Request = Dimelo con flores
  • Error HTTP 401 Unathorized = Estoy casada
  • Error HTTP 402 Payment Required = Son 10.000 y la cama
  • Error HTTP 403 Forbidden/Access Required = ZZZZZZZAS! (bofeton)
  • Error HTTP 404 Not Found = He quedado con las amigas
  • Error HTTP 405 Method Not Allowed = No, por detras no…
  • Error HTTP 406 Method Not Acceptable = … y en la boca menos
  • Error HTTP 407 Proxy Auth. Required = Lo hablare con mi madre
  • Error HTTP 408 Request Timeout = ¿Sabes cuanto hace que no mellamas?
  • Error HTTP 409 Conflict = ¿Quien era ESA?
  • Error HTTP 410 Document Removed = Quiero el D-I-V-O-R-C-I-O
  • Error HTTP 411 Lenght Required = A eso llamas “pedazo polla”?
  • Error HTTP 412 Precondition Failed = ¿Que no has traido condones?
  • Error HTTP 413 Request Entity Too Large = Eso no me va a caber ahi dentro
  • Error HTTP 414 Request URL Too Large = Eso tampoco me va a caber ahidentro
  • Error HTTP 415 Unsupported Media Type = A cuatro patas no me gusta
  • Error HTTP 500 Internal Server Error = Tengo la regla
  • Error HTTP 501 Not Implemented = Nunca haria eso
  • Error HTTP 502 Bad Gateway = … y por ahi, menos todavia
  • Error HTTP 503 Service Unavailable = Me duele la cabeza
  • Error HTTP 504 Gateway Timeout = ¿Ya esta?
